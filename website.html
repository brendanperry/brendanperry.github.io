About

My most recent work is the website that you are currently using right now. I decided to create my own website to get more practice using HTML, CSS, and Javascript.

My old website was made using Wordpress, and compared to this one, it was very slow. This website is a static website that is hosted on GitHub Pages. All of the code is open source on my GitHub.

Although I already knew how to make websites before making this one, I have learned a lot while making it. My biggest areas that I have improved on are making the pages responsive, adding animations, and writing more efficient organized code.

I enjoy having complete control over my own website, so I do not have to rely on any themes or hosting specifically. This project has been really good practice for me to build my web design skills.
